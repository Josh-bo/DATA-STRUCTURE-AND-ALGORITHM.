An array is a data structure that stores a fixed-size sequence of elements of the same type. It provides a way to efficiently store and access multiple values in a single variable. The elements in an array are typically accessed using an index, which represents the position of an element within the array.

FIVE (5) TYPES OF ARRAY METHODS:

1. Length: This method returns the number of elements in an array or list. It allows you to determine the size of the array or list.
2. Index: These methods retrieve the element at a specific index within the array or list. The index starts from 0.
3. Append: These methods add an element to the end of the array or list, increasing its size.
4. Remove: This method removes an element from the array or list at a specified index, reducing its size.
5. Set: These methods modify the element at a given index within the array or list.
